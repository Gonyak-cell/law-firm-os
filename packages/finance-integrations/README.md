# @law-firm-os/finance-integrations

RP23 "External Integrations II" descriptor-only program package for bank feeds, card feeds, WEHAGO/Douzone export, tax export, and DART records. Gates H23/C23, upstream RP22. Every pack in this package starts descriptor-only: no bank/card/accounting/tax/DART runtime, no runtime permission or audit evaluation, no state writes, no object storage, no Hermes runtime receipts, no real financial/client data, no external credentials, no Claude final approval, and no enterprise trust claim from local validation.

## CP00-694

CP00-694 bootstraps the RP23 program: the program contract (contracts/external-integrations-ii-contract.json), this package, and the rp23:external-integrations-ii:validate gate. It covers RP23.P00 Contract And Acceptance Baseline from M00-M10 and opens RP23.P01 Domain Model through M02.S08 as fourteen descriptor-only sections over 150 planned rows. The pack hands off to RP23.P01.M02.S09, with all finance integration runtimes, financial payloads, external credentials, permission decision details, audit bodies, Hermes runtime receipts, and product-state writes closed.
