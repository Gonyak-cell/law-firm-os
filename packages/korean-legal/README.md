# @law-firm-os/korean-legal

RP24 "Korean Legal Depth" descriptor-only program package for HWPX documents, Korean clause references, litigation timelines, corporate filings, court deadlines, and Korean templates. Gates H24/C24, upstream RP23. Every pack starts descriptor-only: no HWPX parse runtime, no court filing runtime, no litigation deadline calculation runtime, no corporate document assembly runtime, no final Korean legal authority claim, no state writes, no object storage, no Hermes runtime receipts, no real client/matter/source payload data, no external credentials, no Claude final approval, and no enterprise trust claim from local validation.

## CP00-722

CP00-722 bootstraps the RP24 program: the program contract (contracts/korean-legal-depth-contract.json), this package, and the rp24:korean-legal-depth:validate gate. It covers RP24.P00 Contract And Acceptance Baseline from M00-M10 and opens RP24.P01 Domain Model through M02.S08 as fourteen descriptor-only sections over 150 planned rows. The pack hands off to RP24.P01.M02.S09, with all HWPX, Korean clause, litigation, court filing, corporate document, permission, audit, Hermes runtime, and product-state surfaces closed.
