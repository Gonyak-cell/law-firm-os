# Amplitude Flow Map

This map groups all numbered screenshots into product flows. It intentionally uses the Amplitude corpus as the planning source, not the existing local showcase HTML.

## marketing-entry (0-0)

- Family: Marketing landing
- Screen type: public landing page with top nav, geometric hero, CTA pair, customer logos, downstream product section
- UI elements: public top navigation, geometric brand motif, primary and secondary CTA, logo trust strip, section preview
- matter destination: Public matter marketing shell and pre-auth brand promise
- Implementation phase: P2 auth and onboarding flow
- Coverage rule: all screenshots in this range must be represented in component states or flow states.
