# Goal Closeout Evidence

Every future Law Firm OS goal should create one folder here:

```text
docs/goal-closeout/{goal_id}/
```

Each goal folder should contain:

```text
packet.json
claude-review-result.json
adjudication.md
construction-inspection.json
command-evidence.json
```

Use `docs/goal-closeout-protocol.md` and `contracts/goal-closeout-protocol-contract.json` as the governing process.
