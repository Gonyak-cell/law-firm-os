# L6 — 운영 체계 수립 (TUW 분해)

분해 대상: `workbook/matter-post-cp-launch-plan.md` §5 L6(:215-227).
공통 프로토콜 §0은 마스터 §F에서 상속하며 본 문서에 복제하지 않는다.
phase 진입조건(L3 모니터링 스택 가동)은 `WP:LT-L3-W06` 의존으로 바인딩했다. L6은 L4·L5와 부분 병렬이며, phase_order상 L5가 선행이므로 L5 TUW에 직접 의존 가능하다.
L6 완료 게이트 4요소의 귀속: 전 runbook 탁상연습 1회 → W01-T03·W03-T02·W05-T02 / 모의 장애 1회 처리 → W01-T03 / AI 리뷰어 명단·권한 부여 → W04-T01 / 변경 1건 처리 시연 → W05-T02. 산출물 '런북 3종' 귀속: incident=W01, break-glass=W03, rollback=W05.
